# Zungo Net - Backend Oficial

Este é o backend do aplicativo Zungo Net — uma rede social inspirada no Facebook, Instagram e WhatsApp.

## Tecnologias
- Node.js
- Express
- Supabase
- API REST

## Como funciona
- Registro de usuários
- Login
- Conexão ao banco de dados Supabase
- API simples para o aplicativo mobile
